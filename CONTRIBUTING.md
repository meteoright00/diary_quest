# コントリビューションについて

このプロジェクトに関心を持っていただき、ありがとうございます！

現在、**diary_quest** は個人の趣味のプロジェクトとして、自分自身の手で世界観を作り上げ、開発を進めることを楽しんでいます。
そのため、大変恐れ入りますが、コードの直接的な変更提案（Pull Request）は現在受け付けておりません。

いただいた Pull Request は、自動的にクローズさせていただく設定となっております。何卒ご了承ください。

### バグ報告・アイデアについて

一方で、**Issues（課題・要望）** への投稿は大歓迎です！
もしバグを見つけたり、「こんな機能があったら面白そう！」というアイデアがありましたら、ぜひお気軽に Issue を立てて教えてください。

皆様の声は開発の大きな励みになります。今後とも diary_quest をよろしくお願いいたします！
