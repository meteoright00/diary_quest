# diary_quest 🗡️📖

RPG風日記アプリケーション - 日常の日記を冒険日誌に変換しよう！

## 📖 概要

**「日記を書く」** それは、あなた自身の冒険の記録です。

diary_questは、あなたの日常をファンタジー世界の冒険譚に変えるデスクトップアプリケーションです。
日々の出来事を日記として記録すると、AIがそれを「冒険日誌」に自動変換。
現実のタスクを「クエスト」として達成し、経験値を獲得してキャラクターを育てましょう。

### ✨ 主な特徴

*   **冒険への変換**: あなたの日常が、壮大な物語の一部になります。
*   **クエスト管理**: 「毎日30分勉強する」「ジムに行く」などの現実の目標をクエストとして管理。
*   **成長レポート**: 週次・月次・年次の活動、感情推移、執筆頻度をグラフで可視化し、AIによる総評を受け取れます。
*   **ストーリーモード**: 指定した期間の日記をまとめ、AIがひとつの壮大な物語（章立てされた冒険譚）として再構成します。
*   **完全プライベート**: データはすべてあなたのPC内に保存されます。AI処理時のみ安全に通信を行います。

### 📊 レポートとストーリー

日常の記録が積み重なると、より深い楽しみ方が可能になります。

*   **成長レポート**: 「レポート」タブでは、書いた文字数や感情の傾向（ポジティブ/ネガティブ）をグラフで確認できます。AIがあなたの活動を分析し、フィードバックをくれます。
*   **ストーリー生成**: 「ストーリー」タブで期間を指定すると、過去の日記をつなぎ合わせて一つの短編小説を生成できます。シリアス、コミカルなど作風も選べます。

## 🎮 利用方法

### 1. アプリケーションの開始

配布されているインストーラーを実行してアプリをインストール・起動します。

### 2. 初期設定（APIキーの設定）

本アプリはAI機能を使用するため、以下のいずれかのAPIキーが必要です：
*   OpenAI (GPT)
*   Anthropic (Claude)
*   Google (Gemini)

設定画面（歯車アイコン）から「LLM設定」を開き、取得したAPIキーを入力してください。
※ APIキーは端末内に暗号化して保存され、外部に送信されることはありません。

### 3. 世界観の選択

「ファンタジー」「SF」「現代ドラマ」など、好みの世界観を選択します。
これによって、日記が変換されるトーンが変わります。

### 4. 冒険（日記執筆）の開始

1.  **日記を書く**: 「日記」タブから、日付・タイトル・今日の出来事を入力します。
2.  **RPG風に変換**: 「変換」ボタンを押すと、AIが内容を分析し、冒険日誌と感情詳細レポートを先に生成・表示します。
3.  **クエスト報告（任意）**: 関連するクエストがある場合、リストから選択して報告できます（保存時に進捗が加算されます）。
4.  **保存と報酬**: 「保存」すると経験値(EXP)とゴールドを獲得。運が良ければ「ランダムイベント」が発生し、アイテムが手に入ることも...？

## 🛡️ プライバシーとセキュリティ

当アプリケーションは**ローカルファースト**で設計されています。
日記データや個人設定がクラウドサーバーに保存されることはありません。
安心してあなただけの冒険を記録してください。
