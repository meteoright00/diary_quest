# ファンタジー世界観設定

## 世界の基本設定
- **世界名**: エルデンワールド
- **時代**: 剣と魔法の中世ファンタジー
- **特徴**: 冒険者ギルドが存在し、モンスター討伐や依頼をこなして生計を立てる世界
- **魔法**: 魔力を使った魔法が一般的に存在する

## あなた（主人公）の設定
- **名前**: [ユーザーが設定]
- **職業**: 駆け出しの冒険者
- **所属**: 王都のギルド「銀の月」
- **特技**: 基礎的な剣術と回復魔法
- **目標**: 一人前の冒険者になり、伝説の遺跡を探索すること

### キャラクターステータス
キャラクターのステータスは `キャラクターステータスサンプル.json` を参照してください。

**主要ステータス:**
- レベル: 日記を書くことで経験値を獲得し、レベルアップ
- HP/MP/スタミナ: 冒険に必要な基本リソース
- 攻撃力、防御力、魔力など: 戦闘や冒険の能力値
- スキル: 特定の条件でアンロックされる特殊能力
- 装備: 武器、防具、アクセサリで能力を強化
- 称号: 実績を達成することで獲得
- クエスト: 達成目標（例：7日連続で日記を書く）

**ステータスの活用:**
- 日記変換時にキャラクターステータスを参照し、より個性的な日記に
- レベルや装備に応じて、日記の表現が変化
- 例：レベル1「駆け出しの冒険者」→レベル10「熟練の冒険者」

## 用語マッピング（現実 → 世界観）

### 場所
- 会社/職場 → ギルド
- オフィス → ギルドホール
- 会議室 → 作戦会議室
- 自宅 → 宿屋の自室
- 通勤電車/バス → 馬車、徒歩
- カフェ/レストラン → 酒場、食堂
- スーパー/コンビニ → 市場、雑貨屋
- 病院 → 教会、治療院

### 人物
- 上司 → ギルドマスター
- 同僚 → 仲間の冒険者
- 部下 → 後輩冒険者
- 取引先 → 依頼主、商人
- 友人 → 旅の仲間
- 家族 → 故郷の家族

### 仕事・活動
- 仕事 → 依頼、クエスト
- 会議 → 作戦会議、冒険者の集会
- プロジェクト → 大規模討伐任務
- 出張 → 遠征
- 残業 → 長時間の探索
- 休暇 → 休息日
- 研修 → 修行、訓練
- プレゼン → 報告会

### 物・道具
- パソコン → 依頼書、地図
- 携帯電話 → 魔法の通信石
- メール/チャット → 伝書鳩、伝令
- お金 → 金貨、銀貨
- 書類 → 羊皮紙、巻物
- 車 → 馬、馬車

### 感情・状態
- 疲れた → 魔力を消耗した、体力が尽きかけた
- ストレス → 冒険の重圧
- 達成感 → 依頼達成の喜び
- 失敗 → クエスト失敗、討伐失敗

## 変換ルール

### 文体
- 一人称視点で、冒険日誌として記述
- やや古風な言い回しを使用（「〜である」「〜せり」など時々混ぜる）
- 臨場感を出すために具体的な描写を追加

### 変換時の注意点
1. 元の日記の感情や出来事の本質は維持する
2. 固有名詞（人名など）は適切にファンタジー風にアレンジ
3. 時間経過や天候などの描写を追加してリアリティを出す
4. 必要に応じてモンスターや魔法の要素を自然に織り込む
5. 過度に大げさにせず、日常的な冒険の記録として描く

```
【世界観】
[上記の世界設定を挿入]

【用語マッピング】
[上記のマッピング表を挿入]

【変換ルール】
[上記のルールを挿入]

【元の日記】
{ユーザーが書いた日記}

【指示】
上記の世界観設定と用語マッピングに従って、元の日記を冒険日誌として自然に変換してください。
感情や出来事の本質を維持しつつ、ファンタジー世界の冒険として魅力的に描写してください。
```

## サンプル変換例

### 元の日記
```
今日は会社で大きなプロジェクトの会議があった。
上司からの指示が曖昧で、同僚たちと何度も確認し合った。
結局、夜遅くまで残業することになり、へとへとになって帰宅した。
明日も早いので早く寝よう。
```

### 変換後
```
本日、ギルドにて大規模討伐任務の作戦会議が開かれた。
ギルドマスターからの指示が不明瞭で、仲間の冒険者たちと何度も確認を重ねる必要があった。
結局、月が高く昇るまで準備に追われ、疲労困憊で宿屋へと戻った。
明朝も早くから訓練があるゆえ、今宵は早めに休むとしよう。
```
